# Mohammed Emran Hossain — Portfolio

Personal portfolio site for **Mohammed Emran Hossain** — SOC Analyst specializing in
security operations, identity & endpoint security, and data analytics.

🔗 **Live:** https://meh1999.github.io

## Stack
Static site (HTML / CSS / vanilla JS) served by **GitHub Pages** — no build step.

```
index.html            # single-page site
assets/css/style.css  # theming (periwinkle + navy palette)
assets/js/main.js     # nav, mobile menu, scroll reveals
assets/img/profile.jpg
```

## Local preview
```bash
python3 -m http.server 8080   # then open http://localhost:8080
```

## Deploy
Push to the `main` branch of `MEH1999/MEH1999.github.io`; Pages serves the root automatically.
